# Arduino-Code

Various code created to use with an Arduino or Arduino-like board.

## Pi Pico Keyboard Jiggler

A script I made for a Pi Pico that will pretend to eb a keyboard when plugged in.  It could be reconfigured to do specific output, but currently just randomly sends space, or WASD commands.  I use it to idle in Fortnite LEGO for experience and avoid the inactivity timeout.
